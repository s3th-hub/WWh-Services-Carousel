A responsive, animated Services Carousel Elementor widget powered by a dedicated Services CPT.

== Description ==

WWH Services Carousel adds:

* A **Services** Custom Post Type (CPT) — manage all your services via the WordPress admin.
* A custom **Elementor widget** (WWH Services Carousel) that renders your services in a beautiful, animated carousel.

**Features**

* Powered by Swiper.js v11 — infinite loop, autoplay, touch/swipe, keyboard navigation.
* Responsive: 3 columns (desktop) → 2 (tablet) → 1 (mobile).
* Hover animations: card lift + scale, image zoom, gradient overlay, button arrow slide.
* Consistent card height via Flexbox + CSS `line-clamp`.
* Full Elementor free-version integration with controls for: query, autoplay, speed, number of slides, colors, typography, shadows.
* Accessible: ARIA labels, keyboard navigation, reduced-motion support, forced-colors mode.
* Google Fonts: **Manrope** (headings) + **Inter** (body).

== Installation ==

1. Upload the `wwh-services-carousel` folder to `/wp-content/plugins/`, or install directly via **Plugins → Upload Plugin**.
2. Activate the plugin.
3. Go to **Services → Add New** to create your first service (add a title, featured image, and excerpt).
4. Open Elementor on any page, search for **"WWH Services Carousel"** in the widget panel, and drag it onto your page.
5. Configure the widget settings and publish.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =
No. The plugin works with the free version of Elementor.

= Where do I add services? =
Navigate to **Services** in the WordPress admin sidebar. Each service supports a title, featured image, short excerpt, and full content.

= Can I control the number of cards shown? =
Yes — use the **Number of Services**, **Slides per View (Desktop)**, and **Slides per View (Tablet)** controls in the Elementor widget panel.
